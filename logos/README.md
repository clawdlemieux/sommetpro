# Sommet Pro — Logo Usage Guidelines

## Logo Variants

| File | Use Case | Size |
|------|----------|------|
| `sommetpro-primary.svg` | Default logo for light backgrounds (website header, documents, email signatures) | 400x100 |
| `sommetpro-stacked.svg` | Square-ish layouts (social media posts, signage, business cards) | 200x180 |
| `sommetpro-icon.svg` | Favicon, profile photos, app icons, small circular placements | 200x200 |
| `sommetpro-dark.svg` | Dark backgrounds, overlays on photos, dark-mode contexts | 400x100 |

## Brand Colors

| Name | Hex | Usage |
|------|-----|-------|
| Deep Forest Green | `#2D4A3E` | Primary brand color, text, front mountain peak |
| Warm Stone | `#C4A882` | Accent, back mountain peak |
| Near Black | `#1A1A1A` | Body text |
| White | `#FFFFFF` | Backgrounds, reversed text/icons on dark |

## Clear Space

Maintain a minimum clear space around the logo equal to the height of the "S" in SOMMET on all sides. No other elements should encroach on this space.

## Minimum Sizes

- **Primary / Dark:** minimum width 200px (digital), 50mm (print)
- **Stacked:** minimum width 100px (digital), 25mm (print)
- **Icon:** minimum 32x32px (favicon), 48x48px (profile images)

## Do's

- Use the SVG files directly for best quality at any size
- Use the dark variant on dark backgrounds or photo overlays
- Use the icon variant for small circular placements
- Convert to PNG at 2x resolution when raster format is required

## Don'ts

- Do not alter the colors of the logo
- Do not stretch, skew, or rotate the logo
- Do not add drop shadows, gradients, or outlines
- Do not place the primary (light) version on dark backgrounds — use the dark variant
- Do not rearrange the icon and wordmark
- Do not use the logo smaller than the minimum sizes listed above

## Typography Note

The wordmark uses Helvetica Neue Bold (with Arial / sans-serif fallback). If embedding in contexts where fonts may not render (e.g., email), export as PNG or use the SVG with embedded outlines.
